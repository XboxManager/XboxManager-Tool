# 🎮 XboxManager-Tool

XboxManager é um utilitário Windows desenvolvido para gerenciamento, transferência de arquivos e ferramentas de configuração para consoles Xbox.

---

## 🚀 Recursos

- 📁 Gerenciamento de arquivos
- 🌐 Conexão FTP com Xbox
- ⚙ Ferramentas de configuração
- 🔄 Verificação de atualização
- 🖥 Interface simples e organizada

---

## 💻 Requisitos

- Windows 7 ou superior
- .NET Framework compatível
- Rede local para recursos FTP
